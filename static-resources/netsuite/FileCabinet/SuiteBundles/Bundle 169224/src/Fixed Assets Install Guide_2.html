Fixed Assets Register - Installation Guide

Document Overview:

This document provides only basic installation instructions - for full user documentation please contact your NetSuite supplier.

Installation Instructions:

Login as the Administrator user and navigate to Fixed Assets, Setup, System Setup, complete the required settings and then click Submit; the settings can be changed later if required, but the System Setup record must be generated before other features of the system can be used.

At least one Asset Type should be entered, since depreciation and reporting depend upon asset types and therefore every asset should have an asset type.  To create an Asset Type, navigate to Fixed Assets, Setup, Asset Types.
